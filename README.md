### Technologies

- ⭐ CSS **Grid** Layout
- ⭐ modern CSS + SASS (SCSS)
- ⭐ **Responsive Design** - mobile, tablet and Desktop compatible.
- ⭐ **block__element--modifier** pattern.
- ⭐ npm **compile, prefix & compress**.
- ⭐ **SVG** icons.

### Features
- ⭐ **Animations**
- ⭐ responsive, **Foldable menu**
- ⭐ **mark liked houses**
- ⭐ **gallery**
